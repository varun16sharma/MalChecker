About My Project

It is a web application that classifies the files as malicious or non-malicious with the help of machine learning.

It tries out various classification algorithms which is used for prediction and comparing the results.

Classification algorithms that are tried are RandomForest, DecisionTree, Adaboost, Gaussian and Gradient Boosting.

In order to test the model for file, it's required to extract the characteristics of the given file.

ML model is used to predict the class for the given file based on the already trained model.
